## Overview

- **Cross-platform** — one native app for Android, iOS, macOS and Windows, built with Flutter.
- **Local-first** — data lives on the device. No account, no server, no tracking. Backups are portable export files the user controls.
- **Native PDF output** — programs are rendered directly to print-ready PDF with an adaptive column layout; the preview matches the printed result exactly.
- **Auto-updated workbooks** — meeting workbooks are downloaded and cached automatically, so new weeks are available as soon as they are published.
- **Multilingual** — the interface ships in Spanish and English, independent of the meeting language.

## Who Agora is for

Agora is made for those who prepare the midweek meeting program of their congregation and spend evenings adjusting tables in a word processor to do it. If you schedule assignments, balance participants and print the weekly program, Agora is for you.

It starts with the midweek meeting and is growing into a general program generator for the congregation: more program types are planned on the same foundation (projects, participants, congregations).

## Features

- **Program builder** — select the weeks and Agora lays out every part of the meeting with sections, songs and timing already in place.
- **Automatic timing** — start time, section durations, student counsel minutes and slack are computed automatically; every row gets its exact time.
- **Participants directory** — manage the congregation's participants with their privileges (elder, ministerial servant, publisher), so the right names are offered for the right parts.
- **Projects dashboard** — group weeks into projects (a full month or a single week), track their status from draft to exported, and keep reminders at hand.
- **Auxiliary classroom** — enable a second classroom and the program renders both side by side, with adaptive columns that never overflow.
- **Circuit overseer visits** — mark the week and the program adapts automatically: the Bible study is replaced by the overseer's talk and timing is recalculated.
- **Editable titles** — rename any assignment inline; custom titles stay in sync between the editor and the printed PDF.
- **Live preview** — the final PDF re-renders in real time as each name is typed, removing the export-and-check loop.
- **One-tap export** — save or share the finished PDF directly from the app.
- **Light and dark theme** — both modes fully supported, following the system or the user's choice.

## Does Agora need the cloud?

No. Agora works in local mode: everything is stored on the device and the app is fully functional offline. The only network use is downloading meeting workbooks when new ones are published. Portable backup and restore keeps data safe and allows moving between devices.

## Tech stack

- [Flutter](https://flutter.dev) + [Riverpod](https://riverpod.dev) — UI and state management
- [pdf](https://pub.dev/packages/pdf) + [pdfrx](https://pub.dev/packages/pdfrx) — native PDF generation and real-time preview
- [slang](https://pub.dev/packages/slang) — type-safe i18n (Spanish, English)
- [Drift](https://drift.simonbinder.eu) + encrypted storage — local database foundation

## Getting started (development)

```bash
git clone https://github.com/vnevarezt/Agora.git
cd Agora
flutter pub get
flutter run          # pick your device: macOS, Windows, Android, iOS
```

Useful commands:

```bash
dart run slang       # regenerate translations after editing lib/i18n/*.i18n.json
flutter test         # run the test suite
flutter analyze      # static analysis
```
